# border_blur

create border blur without center area

## Params
 - Border Radius
 - Padding
 - Blur value
## Usage

Container(
    width: 500,
    height: 300,
    child: BorderBlur(
        blur: 2.0,
        padding: 30.0,
        radius: radius,
    ),
)
